# Artwork

Design source files for Remini Labs. This folder holds **Tier 2** assets — the
canonical originals that runtime (`static/`) derivatives are generated from.

## Layout

```
artwork/
  brand/                  logo, wordmark, palette, typography samples
  (illustrations/)        future: illustrations, patterns
  (icons/)                future: custom glyphs beyond Lucide
  (mockups/)              future: UI concepts, design explorations
  (marketing/)            future: OG card templates, email headers
```

Parentheses = placeholder for future folders. Create them only when content
arrives — don't pre-commit empty directories.

## Tiers

| Tier | Folder                   | What it is                                            | Lifecycle             |
| ---- | ------------------------ | ----------------------------------------------------- | --------------------- |
| 1    | (external — Figma cloud) | Editable design source (.fig)                         | Edited in design tool |
| 2    | `artwork/`               | Committed design exports (.svg, .png master)          | Committed to git      |
| 3    | `static/`                | Runtime-served derivatives (favicon, og-\*, manifest) | Generated from Tier 2 |

Not every project needs Tier 1 — solo projects often start at Tier 2 with a
raster or hand-drawn SVG and no upstream Figma file.

## Naming conventions

- **Lowercase, hyphen-separated**: `logo-mark-2560.png`, never `LogoMark2560.png`.
- **Name by intent, not by location of use**: `logo-mark.svg`, not `header-logo.svg`.
- **Size suffix only for rasters** (vectors have no size): `logo-mark-2560.png`,
  but `logo-mark.svg` stays unsized.
- **Role suffix when there are variants**: `-master`, `-source`, `-inverted`,
  `-mono`.
- **No version suffixes in filenames** (use git tags instead) — never
  `logo-v2-final.png`.

## Relationship to `static/`

Files in `static/` are **regenerable** from `artwork/`. The canonical
generator is `scripts/gen-brand-assets.mts` (run via `pnpm gen:brand-assets`).
If a `static/` icon is lost or needs updating, regenerate from the master
rather than editing the runtime file directly.

Files in `static/` that are **NOT** derived from `artwork/` and ARE
hand-authored:

- `safari-pinned-tab.svg` — monochrome R glyph, hand-authored SVG
- `manifest.webmanifest` — PWA manifest, hand-authored JSON
- `og-default.png`, `og-weather.png`, `og-tickers.png` — OG social cards
  generated by screenshotting a temporary Svelte route (see git history for
  the `_og/[name]/+page@.svelte` pattern)
